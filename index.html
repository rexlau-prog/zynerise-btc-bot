Methodology & Verification

All trades execute against real-time Binance BTCUSDT perpetual prices. Each entry timestamp + entry price below can be cross-referenced against Binance's historical 1-minute candles for that minute (or against TradingView chart BINANCE:BTCUSDT.P). Stop-losses and take-profit levels also fire on Binance perp prices.

Starting capital: $10,000. Position sizes are determined by each plan's tier weight × the bot's max-position cap. Realized PnL is computed as (exit_price − entry_price) × quantity per partial close, summed over all TP hits and the final SL/TP/manual close.

The equity curve is plotted at each trade's entry time (the trade's full eventual realized PnL is added at that point). This is standard for strategy track-records and emphasizes the trade-by-trade outcomes; it is not a tick-by-tick mark-to-market curve.

APR is annualized linearly from the data window. Short windows extrapolate aggressively — treat APR as a directional signal, not a forward forecast. The same plans also run concurrently on Hyperliquid via a separate execution bot.
